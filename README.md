# 📐 Анализатор Чертежей CAD Pro v24.0

Профессиональный анализатор DXF-чертежей для расчета длины реза на станках ЧПУ и лазерной резки.

## ✨ Основные возможности

- ✅ Расчет длины реза по всем типам объектов (LINE, ARC, CIRCLE, POLYLINE, LWPOLYLINE, SPLINE, ELLIPSE)
- ✅ **Правильный подсчет врезок с анализом связности контуров**
- ✅ Визуализация чертежа с индикацией ошибок
- ✅ Визуализация цепей связанных объектов
- ✅ Статистика по типам объектов и цветам
- ✅ Экспорт результатов в CSV
